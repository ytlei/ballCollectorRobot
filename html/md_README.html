Ball Collector Robot project

Overview

build a simulation of the ball collector robot on ROS. It will be built with turtlebot robot in ROS. The ball collector robot can find the nearest ball and goes to the ball and push them back to the corner for users to collect easily.

The Robot: Use turtlebot as a foundation and use the sensor on it to see the environment and objects. After Acquiring the object, generate a path using navigation technique to collecting the ball. The environment: create a simulated tennis court with balls placed randomly and with walls on four sides. Tools may include Gazebo Simulator, Turtlebot module, and ROS.
